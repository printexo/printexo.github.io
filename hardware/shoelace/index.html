    Shoelace Fixation Guide

    This guide describes how to mount the dorsal interface onto the shoe and secure it using auxiliary shoelaces for PrintExo.

    1. Overview

    The shoelace fixation method provides a non-invasive, shoe-agnostic interface that allows PrintExo to be mounted on standard lace-up footwear without permanent modification.

    This guide covers: - mounting the dorsal interface onto the shoe, - routing and tensioning shoelaces for secure fixation.
