# Quick Start

This page explains **how to navigate the PrintExo documentation** and where to begin depending on your goal.

---

## Documentation Structure

The documentation is organized into the following sections:

- **Build the Robot**  
  fabrication, hardware, and mechanical assembly

- **Electronics & Control**   — *coming soon*  
  example electronics setups and reference control pipelines

- **Troubleshooting**   — *coming soon*  
  common build and operation issues

- **Paper & Citation**  
  research background and academic reference

---

## Where Should I Start?

### If you want to build the device

1. Bill of Materials (BOM)  
2. 3D Printing Guide  
3. Assembly Guide
4. Shoelace Fixation Guide

---

### If you want example electronics or control

Refer to **Electronics & Control** for reference implementations.  
These examples are **not prescriptive** and may be adapted or replaced.

---

### If you are using PrintExo for research

See:

- Build the Robot — Overview  
- Paper & Citation  

---

## Next Steps

Most users should start with:

**Build the Robot → Bill of Materials (BOM)**

and proceed step-by-step through the build guides.
